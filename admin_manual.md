## 1. 登入與權限管理

* **後台網址**：`您的網址/admin` (例如：`https://esgwd.vercel.app/admin`)
* **管理員預設帳密**：
  * **帳號**：`admin`
  * **密碼**：`esgadmin123`

### 操作步驟：
1. 開啟瀏覽器輸入後台網址。
2. 於登入畫面輸入帳號與密碼，點選「登入」。
3. 登入成功後將會看到管理後台的主介面。若想離開，點選側邊欄底部的 **「登出系統」** 即可。

## 2. 系統架構與技術特點

本系統採用了現代化的無伺服器架構，具有以下幾點優勢：

* **雲端與離線雙重同步 (Supabase + localStorage)**：
  所有的內容修改都會即時儲存至 Supabase 雲端資料庫。如果網路中斷，系統會自動切換為本機 LocalStorage 儲存，並在網路恢復時自動重新整理，確保管理工作不中斷。
* **Cloudinary 自動圖床整合**：
  後台各處的圖片上傳按鈕皆與 Cloudinary 雲端整合。當您點擊「上傳圖片」並選擇檔案後，系統會將圖片上傳至雲端儲存，並自動將網址寫入資料庫，您無需手動上傳圖床並複製貼上連結。
* **跨分頁即時同步技術**：
  當您在後台按下「儲存設定」後，所有使用者開啟之前端網頁會**即時自動更新內容**，無需使用者手動點選瀏覽器重新整理。

## 3. 各功能模組操作指南

登入後，您可以使用左側側邊欄在不同管理分頁之間切換。

### 3.1 控制台首頁 (Dashboard)
* **功能**：顯示目前網頁的整體狀況統計。
* **指標**：包含目前活躍的便當格數量、輪播大圖張數、已發布的最新消息文章總數、以及合作夥伴 LOGO 數量。
* **用途**：供快速檢視目前網站的內容規模。

### 3.2 Hero 首頁大輪播管理
此區塊控制首頁頂部最醒目的 3 張大輪播廣告看板。
1. 點選左側側邊欄的 **「首頁大輪播」**。
2. 系統會列出 **Slide 1**、**Slide 2**、**Slide 3** 的編輯區。
3. 可修改項目：
   * **主標題**（例如：AI 音樂跨界創作競賽）
   * **副標題 / 描述文字**
   * **按鈕文字**（例如：立即報名參賽）
   * **按鈕顏色**（可使用調色盤挑選，支援透明度）
   * **背景底圖**：
     * **方式 A（推薦）**：點擊「上傳新背景底圖至 Cloudinary」按鈕，選擇電腦中的圖片，系統上傳完成後會自動更新下方連結。
     * **方式 B**：直接在「底圖連結」欄位貼上外部的圖片 URL。
4. 修改完成後，點擊最下方的 **「儲存 Hero 輪播設定」** 按鈕即可生效。

### 3.3 核心專案管理
控制首頁「核心專案」區塊的 3 張核心計畫卡片（如：永續綠色學校、企業永續發展論壇等）。
1. 點選左側側邊欄的 **「核心專案管理」**。
2. 選擇您要修改的專案卡片，可修改欄位：
   * **專案標題** 與 **專案描述**。
   * **統計數據指標**：例如將 `100+` 所學校、`10,000+` 參與人次的數據即時調升。
   * **數據說明標籤**：統計數字下方的說明文字。
   * **按鈕文字** 與 **卡片圖示**（支援輸入 Lucide 圖示名稱，例如 `BookOpen`, `Users`, `Award`）。
3. 修改完畢後，點擊下方 **「儲存核心專案設定」**。

### 3.4 最新消息管理
管理首頁的最新消息清單，支援前台按分類（全部、企業論壇、學校教育、音樂會）進行篩選。
* **新增消息**：
  1. 點選側邊欄的 **「最新消息管理」**。
  2. 在下方「新增消息文章」區塊填寫：
     * 選擇 **消息分類**。
     * 輸入 **消息標題** 與 **消息摘要**。
     * 填寫 **發布日期**。
     * 上傳配圖：點選「上傳最新消息配圖至 Cloudinary」按鈕，選擇您的新聞封面圖。
  3. 點選 **「發布新消息文章」**，該文章便會立刻顯示於前端最新消息清單中。
* **刪除消息**：
  1. 在「現有消息文章列表」中，找到您要刪除的項目。
  2. 點選該文章右側的 **「刪除」** 按鈕即可移除。

### 3.5 合作夥伴與常青得獎名單
此區塊控制首頁底部的「合作夥伴 Logo 跑馬燈」與「常青得獎名單金卡區」。
* **合作夥伴 LOGO 管理**：
  * **新增合作夥伴**：輸入廠商的「中文名稱」、「英文名稱」，並填寫或上傳 LOGO 圖示網址，點擊「新增合作夥伴」。
  * **刪除合作夥伴**：在列表中點選紅色的「刪除」按鈕。
* **常青得獎名單**：
  * 支援直接修改得獎企業名稱、獎項描述、代表圖示。修改後點擊 **「儲存合作夥伴與得獎設定」**。

### 3.6 電子書與捐款方案管理
管理「出版品」Ebook 下載以及「支持我們」的 Laurel 會員捐款 Tier。
* **電子書管理**：可修改電子書標題、下載按鈕文字，以及 3 個電子書亮點卡片的內容與圖示。
* **捐款方案管理**：
  1. 系統列出三個會員等級（Guardian 守護者、Promoter 推廣者、Navigator 領航者）的設定區。
  2. 可修改項目：方案名稱、每月捐款金額（如 NT$ 500）、滿額禮品名稱與描述。
  3. **會員權益清單**：請使用**「換行」**來分隔每項權益，系統會自動在前端渲染成漂亮的打勾清單。
3. 修改完成後，點擊頁面底部的 **「儲存出版品與捐款設定」**。

### 3.7 便當格區塊排版 (Bento Grid)
此區塊允許管理員自訂 Bento 排版網格的版面。
1. **新增區塊**：填寫標題、內容、選擇尺寸（1x1 迷你、2x1 橫向、2x2 大型寬幅），並設定底色與文字顏色，點擊「創建新 Bento 區塊」。
2. **刪除區塊**：可在下方列表中直接刪除不需要的便當網格。

## 4. 系統維護與設定

如果您在測試期間修改了許多資料，希望在正式上線前將所有網站內容回復至最乾淨的初始狀態，您可以使用重置功能：
1. 點選側邊欄的 **「系統設定」**。
2. 找到 **「回復出廠預設值 (Factory Reset)」** 區塊。
3. 點擊 **「執行重置」** 按鈕。
4. 系統將會清空現有資料，並重新寫入我們預先設定好的精美初始範本內容。

## 5. 上傳圖片建議規範

為了確保網站載入速度最快、版面最精美，請管理員在準備上傳的圖片時遵循以下建議：

| 欄位區塊 | 建議尺寸 (寬 x 高) | 建議格式 | 說明 |
| :--- | :--- | :--- | :--- |
| **首頁大輪播底圖** | 1920 x 1080 像素 | JPG 或 WebP | 盡量選擇暗色調、乾淨的背景，以突顯前景的白色文字。 |
| **最新消息封面圖** | 800 x 600 像素 (4:3) | JPG 或 PNG | 建議主體置中，維持統一比例前台排版會最整齊。 |
| **合作夥伴 LOGO** | 400 x 200 像素 | PNG (去底透明) | 強烈建議上傳背景透明的 PNG，跑馬燈視覺效果最佳。 |
| **電子書封面底圖** | 1200 x 800 像素 | JPG | 供 3D 書本厚度模擬效果使用。 |
